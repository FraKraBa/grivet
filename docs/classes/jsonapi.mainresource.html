Class MainResource

A main resource contained in the data member of the top level JSON:API document.

Always contructed non-lazy from the parent document.

throws

IdMismatchError when the optional id argument does not match the id present in rawData

throws

SchemaError when rawData does not look like a JSON:API resource object
